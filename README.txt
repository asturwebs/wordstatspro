Analiza y muestra estadísticas detalladas de tus posts: número de palabras, tiempo de lectura e índice de legibilidad Fernández-Huerta.

== Description ==

WordStatsPro es una herramienta profesional para analizar y mostrar estadísticas detalladas de tus posts en WordPress. Especialmente diseñado para contenido en español, incorpora el índice de legibilidad Fernández-Huerta, proporcionando información valiosa sobre la complejidad de tu contenido.

= Características Principales =

* Contador de palabras preciso
* Cálculo del tiempo estimado de lectura
* Índice de legibilidad Fernández-Huerta
* Información detallada sobre el nivel académico del texto
* Interfaz personalizable
* Tooltip informativo con detalles sobre el nivel de legibilidad
* Totalmente responsive

= Niveles de Legibilidad =

El plugin analiza tu contenido y lo clasifica en los siguientes niveles:

* 90-100: Muy fácil (Primaria baja)
* 80-89: Fácil (Primaria media)
* 70-79: Algo fácil (Secundaria baja)
* 60-69: Normal (Secundaria alta)
* 50-59: Algo difícil (Bachillerato)
* 30-49: Difícil (Nivel universitario)
* 0-29: Muy difícil (Postgrado y profesionales)

= Personalización =

* Ajusta la velocidad de lectura (palabras por minuto)
* Personaliza colores de fondo y texto
* Elige entre diferentes tipos de fuente
* Modifica el tamaño del texto

== Installation ==

1. Sube la carpeta `wordstatspro` al directorio `/wp-content/plugins/`
2. Activa el plugin a través del menú 'Plugins' en WordPress
3. Ve a 'Ajustes' > 'WordStatsPro' para configurar las opciones

== Frequently Asked Questions ==

= ¿Cómo se calcula el tiempo de lectura? =

El tiempo de lectura se calcula dividiendo el número total de palabras entre la velocidad de lectura configurada (por defecto 200 palabras por minuto).

= ¿Qué es el índice Fernández-Huerta? =

Es una adaptación al español del índice Flesch-Kincaid, específicamente diseñado para evaluar la legibilidad de textos en español.

= ¿Puedo personalizar dónde aparecen las estadísticas? =

Actualmente las estadísticas aparecen automáticamente al inicio de cada post. Próximamente se añadirán más opciones de posicionamiento.
